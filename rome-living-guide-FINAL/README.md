# 🏛️ Where Should You Live in Rome?

Interactive neighborhood guide with real rent data from immobiliare.it (Jan 2026).

**Live:** [pvjustin16.github.io/rome-living-guide](https://pvjustin16.github.io/rome-living-guide)

## Features

- 13 neighborhoods with real €/m² rent data
- 5 housing types (Room → 3-Bed) with Italian rental terms
- 4 transport modes with different commute times and costs
- 13 destinations (7 universities + 6 work areas)
- Dual destination support (work + university)
- True Cost sorting (factors in commute time value)
- Expanded details: hospitals, ethnic stores, jobs, groceries, move-in costs

## Data Sources

immobiliare.it · idealista · Google Maps · ATAC · ISTAT

## Tech

Pure vanilla HTML + CSS + JavaScript. Zero dependencies. Zero build step.
